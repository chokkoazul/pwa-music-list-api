## Development

### Dependencies
- Node Version Manager (NVM)

### Installation
Install nvm and then use the default version for the project

```sh
$ nvm use
```

```sh
$ npm install
```

### Run testing

```sh
$ npm run test
```

### Run api

```sh
$ npm run start
```

## Deploy
Deploy on heroku through travis CI
